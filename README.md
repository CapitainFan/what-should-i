Technologies

Backend :
  Node.JS - Epress with TS
  MondoDB - mongoose library
  JWT - access and refresh tokens
  Multer - file manage library
  WebSocket server
  Redis
  N8N agent

Frontend :
  Next with TS
  Tailwind CSS
  Axios
  ShadcnUI
  React-Toastify
